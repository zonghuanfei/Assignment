一，标题  
-----------

标题方式一：一级标题下用“=”，二级标题下用“-”分隔开。  
标题方式二：用n个“#”前置表示n级标题。

## 二，段落

换行：行末添加两个空格来换行  
或两行之间隔一行表示换行

## 三，字体效果

删除字：用两个“~~”夹住字体，效果：“~~字体~~” 

斜体字：用两个“*”夹住字体，效果：“*字体*”   

粗体字：两个“**”夹住字体，效果：“**字体**”

粗斜字体：两个“***”夹住字体，效果：“***字体***”  

## 四，分割线

三个以上的“*“独占一行   

如“****”

****

## 五,引用

> 引用内容

>

> >

机器人学第一章和第二章2.1  
==============

学习知识点概括：
--------------

1.刚体的运动状态如何描述？  
平面上：沿x，y方向位移（2 DOF）＋旋转方向（1 DOF）  
空间中：沿x，y，z方向移动（3 DOF）＋绕x，y，z转动（3 DOF）  
2.如何表达刚体的位移和转动？  
建立frame（常在质心上建立）  
移动由bodyframe的原点位置确定，转动由bodyframe的姿态确定
